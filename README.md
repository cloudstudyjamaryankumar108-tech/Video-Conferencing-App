# Zoomify -- Video Conferencing App 🎥

A modern **Video Conferencing Web Application** built using the **MERN
Stack** to enable seamless virtual communication, collaboration, and
secure online meetings.

## 🚀 Features

### 🎥 High Quality Video & Audio

Provides smooth high-definition video and crystal clear audio for
seamless communication.

### 🖥 Screen Sharing

Participants can share screens, presentations, documents, and other
media files during meetings.

### 🎭 Virtual Backgrounds

Users can customize their meeting environment with virtual backgrounds
and filters.

### 🧠 Interactive Whiteboard

Allows users to draw, brainstorm, and collaborate in real-time on a
shared whiteboard.

### 🎙 Meeting Recording

Meetings can be recorded and stored for later reference.

### 🔒 Secure Communication

-   End-to-end encryption\
-   Password protected meetings\
-   Waiting room security

### 📱 Cross Platform Support

Works on: - Windows - macOS - Linux - Android - iOS

## ⚡ Installation

### 1️⃣ Clone the repository

``` bash
git clone https://github.com/cloudstudyjamaryankumar108-tech/Video-Conferencing-App.git
cd Video-Conferencing-App
```

### 2️⃣ Install dependencies

``` bash
npm install
```

### 3️⃣ Run the application

``` bash
npm start
```
